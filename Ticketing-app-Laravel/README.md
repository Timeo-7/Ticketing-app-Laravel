# Ticketing App Laravel

## Description

**Ticketing App Laravel** est une application de gestion de tickets, projets et clients, développée avec **Laravel 12** et **PHP 8.2**. Elle fournit une interface sécurisée par authentification pour :

- Créer, modifier et valider des tickets de support
- Administrer des projets et des clients
- Suivre les statuts et l’avancement des tâches

---

## Fonctionnalités

- Authentification utilisateur : connexion, inscription, gestion du profil
- Tableau de bord personnalisé après connexion
- Gestion complète des tickets : création, édition, suppression, validation
- Gestion des projets : création, édition, suppression, validation
- API REST pour les opérations CRUD sur les tickets
- Migrations et seeders pour initialiser et peupler la base de données

---

## Stack technique

- **PHP** ^8.2  
- **Laravel** ^12.0  
- **Laravel Sanctum** (authentification API)  
- **Laravel Breeze** (authentification user)  
- **Vite** (build frontend)  
- **Tailwind CSS** (UI)  
- **Alpine.js** (interactivité légère)  
- **Axios** (requêtes HTTP)  
- **PestPHP** (tests unitaires et fonctionnels)

---

## Prérequis

- PHP 8.2 ou supérieur  
- Composer  
- Node.js et npm  
- Base de données compatible : MySQL, SQLite, PostgreSQL, etc.
